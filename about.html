This is a ticketing suite which is also called an ITS or Issue Tracking System. This is used to maintain and keep track of issues pertaining to a certain topic or combination of topics. This project is maintained by Chris Marotta whom you can find on github at click me . This was a project originally created so that I could learn how to use html and make websites but also to create a strong but also user friendly ticketing system that can be used in a multitude of different environments!




PLEASE NOTE:

This software suite is still very much a work in progress so please report any bugs on the github page at here and please be patient as I am only one person whom is working very hard to try to develop this!
